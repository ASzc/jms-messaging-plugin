Create task repos for the specified task IDs.

For each task ID specified an environment variable is created named TASK_REPO_URLS_<task-id>. This variable contains a semicolon-separated list of repo URLs for the specified task ID.

Additionally, the environment variable TASK_REPO_URLS is created containing a semicolon-separated list of all the repo URLs for all the task IDs.

These values are also stored in the file task-repo.properties in the workspace.
